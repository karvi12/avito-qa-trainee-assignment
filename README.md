# QA Internship Assignment – Avito Tech (Spring 2026)

Репозиторий содержит решение тестового задания на стажировку QA в Avito.  
Выполнены: тест-кейсы и автоматизированные проверки для UI (десктоп + мобильная версия).  
Тесты написаны на **Python** с использованием **Playwright** и **pytest**.

## Что сделано

- Составлены тест-кейсы для всех заявленных сценариев десктопной и мобильной версий.
- Автоматизированы UI-тесты:
  - Фильтр по диапазону цен (позитивные и негативные проверки, граничные значения).
  - Сортировка по цене (возрастание / убывание).
  - Фильтр по категории.
  - Тогл «Только срочные».
  - Управление таймером на странице статистики.
  - Переключение тёмной / светлой темы в мобильной эмуляции.
- Найденные дефекты задокументированы в `BUGS.md`.
- Все тесты независимы, воспроизводимы и запускаются одной командой.

## Стек технологий

- Python 3.10+
- Playwright (синхронное API) – автоматизация браузера
- pytest – запуск тестов и фикстуры
- Allure (опционально) – формирование отчётов

## Быстрый старт

1. Клонировать репозиторий
git clone https://github.com/karvi12/avito-qa-trainee-assignment.git
cd avito-qa-trainee-assignment

2. Создать и активировать виртуальное окружение

Windows:
python -m venv venv
venv\Scripts\activate

macOS / Linux:
python3 -m venv venv
source venv/bin/activate

3. Установить все зависимости

pip install -r requirements.txt
playwright install chromium

4. Запустить тесты

5. # Все UI-тесты
pytest -v

# Только десктопные сценарии
pytest test_desktop.py -v

# Только мобильные сценарии
pytest test_mobile.py -v

Для наблюдения за выполнением браузер запускается в видимом режиме (headless=False).
Чтобы запустить в фоне, измените в conftest.py параметр headless=True.
