# Тест-кейсы

## TC-1: Проверка работы фильтра "Диапазон цен"

**Окружение:**
- Браузер: Chromium (Playwright) / Chrome последней версии
- ОС: Windows 11
- Дата: 08.04.2026
- Стенд: https://cerulean-praline-8e5aa6.netlify.app/

**Приоритет/Критичность:**
P1 (высокий)

**Тестовые данные:**
«От» = 1000, «До» = 5000

**Шаги:**
1. Открыть главную страницу https://cerulean-praline-8e5aa6.netlify.app/
2. В поле "От" ввести 1000
3. В поле "До" ввести 5000
4. Дождаться автоматического применения фильтра (изменение URL или перерисовка карточек).
5. Проверить цены отображаемых объявлений. 

**Ожидаемый результат:**
- URL страницы содержит параметры minPrice=1000 и maxPrice=5000.
- Все отображаемые объявления имеют цену в диапазоне от 1000 до 5000 включительно.
- Отсутствуют объявления с ценой меньше 1000 или больше 5000.

## TC-10: Проверка управления таймером на странице статистики

**Окружение:**
- Браузер: Chromium (Playwright) / Chrome последней версии
- ОС: Windows 11
- Дата: 08.04.2026
- Стенд: https://cerulean-praline-8e5aa6.netlify.app/

**Приоритет:** P2 (средний)

**Предусловие:**
- Страница статистики доступна по адресу `/stats`.

**Шаги:**
1. Перейти на страницу `/stats`.
2. Нажать кнопку «Обновить».
3. Убедиться, что индикатор времени последнего обновления изменился (показывает «только что»).
4. Нажать кнопку «Стоп».
5. Запомнить время последнего обновления и подождать 3 секунды.
6. Убедиться, что время обновления не изменилось.
7. Нажать кнопку «Старт».
8. Подождать 3 секунды и убедиться, что время обновления изменилось.

**Ожидаемый результат:**
- Кнопка «Обновить» вызывает обновление данных и сброс таймера.
- Кнопка «Стоп» останавливает автоматическое обновление.
- Кнопка «Старт» запускает автоматическое обновление.

## TC-11: Проверка переключения между тёмной и светлой темой (мобильная версия)

**Окружение:**
- Эмуляция: iPhone 12 (390×844, мобильный User-Agent)
- Браузер: Chromium (Playwright)
- ОС: Windows 11
- Дата: 08.04.2026
- Стенд: https://cerulean-praline-8e5aa6.netlify.app/

**Приоритет:** 
P2 (средний)

**Шаги:**
1. Открыть главную страницу в мобильном разрешении.
2. Определить текущую тему страницы по атрибуту `data-theme` у элемента `html` (`light` или `dark`).
3. Нажать на кнопку переключения темы (класс `._themeToggle_127us_1`).
4. Дождаться изменения атрибута `data-theme`.
5. Убедиться, что значение `data-theme` изменилось на противоположное.
6. Повторно нажать на кнопку переключения темы.
7. Убедиться, что значение `data-theme` вернулось к исходному.

**Ожидаемый результат:**
- При каждом клике атрибут `data-theme` меняется с `light` на `dark` или наоборот.
- Визуальное оформление страницы соответствует выбранной теме.
